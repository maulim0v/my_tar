# my_tar

## Command
./my_tar

## Description
Simple implementation of tar

## Return
0 on success, and >0 if an error occurs.

## Resources
https://www.gnu.org/software/tar/manual/html_node/Standard.html

## Functions used
```
chmod
close
free
fstat
lseek
malloc
mkdir
open
opendir
read
readdir
readlink
stat
symlink
write
```
## Run
```
make
./my_tar
```
